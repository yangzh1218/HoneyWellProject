step 1: check the device

Ensure the device powered on,and near the equipment to perform the add operation.

step 2: scan QR code

  1. Click on the APP home page upper right corner of the "+" enter the scanner page.
  2. Scanning device's QR code in a rectangular box,the system will automatically recognize it.

step 3:Add device

  1. New device will be automatically bound to your account after you configure the WiFi.
  2. Already bound by others the device will prompt you to request authorization.

step 4:Configure WiFi

  1. WiFi configuration process, you need to keep your wireless network is available.
  2. Probably needs 30-60 seconds configuration process, test network environment decides.
  3. Waiting for the bottom of the camera after the WiFi led is green light,indicates the network configuration was successful.
  4. After successful configuration the automatic jump to the device list page.
